【Jekyll入門】Jekyllを用いた個人サイトの構築方法を解説

この記事ではJekyllの導入について解説しています．

By Tomohiro Watanabe
4 min read

はじめに

Windows環境でJekyllを使ってみようとしたら，エラー祭りになって解決に苦戦したため，同じ悩みを持つ方のためにWindows環境でJekyllを使う方法について解説します．

この記事でわかること

Windows環境におけるJekyllの導入方法．

この記事の対象者

この記事は下記の人を対象にしています．

  • WindowsユーザかつJekyllを使ってみたい人

開発環境

OS
Windows 11 Home 25H2
言語
Ruby 3.2.10
ツール
Gem: 3.4.19
Bundler: 4.0.7
Jekyll: 4.4.1

Rubyのインストール

JekyllはRuby製なので，プログラミング言語の一種であるRubyをここからローカル環境に入れます． サイドバーの”WITH DEVKIT”にある項目の中から， ご自身の計算機に合ったものをダウンロードしてください．

例えば，設定 > システム > バージョン情報のデバイスの仕様の欄がシステムの種類　64 ビット オペレーティング システム、x64 ベース プロセッサの場合は，末尾に(x64)がついているものをダウンロードしてください．

テーマとしてChirpyを用いる場合は，Ruby+Devkit 3.2.10-1 を選択します． どうやら，CHirpyは最新バージョンには対応していないようです．

ダウンロードしたインストーラを起動し，画面に表示される指示に適宜答えていってください． 指示に答え終わるとインストールが始まります． 最後にコマンドプロンプトが起動すると思いますので， 1,2,3と入力し，Enterを押してください．

完了後，以下のコマンドをコマンドプロンプトへ入力し， バージョンが出力されることを確認してください．

1
2

$ ruby -v
-> ruby 3.2.10 (2026-01-14 revision a3a6d25788) [x64-mingw-ucrt]

1
2

$ gem -v
-> 3.4.19

ツールのダウンロード

最後に，以下のコマンドを実行しツールをダウンロードしてください．

1

$ gem install jekyll bundler

ダウンロード後に，以下のコマンドを実行し，ダウンロードされたことを確認してください．

1
2
3
4

$ bundle -v
-> 4.0.7
$ bundle exec jekyll -v
-> jekyll 4.4.1

おわりに

Jekyllの導入についてまとめました． 次回は具体的にテーマを使用する方法を解説しようと思います．
