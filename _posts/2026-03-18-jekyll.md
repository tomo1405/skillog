## Rubyのインストール
---
JekyllはRuby製なので，プログラミング言語の一種であるRubyを[ここから](https://rubyinstaller.org/downloads/)ローカル環境に入れます．
サイドバーの"WITH DEVKIT"にある項目の中から，
ご自身の計算機に合ったものをダウンロードしてください．
> 例えば，**設定 > システム > バージョン情報**のデバイスの仕様の欄が`システムの種類　64 ビット オペレーティング システム、x64 ベース プロセッサ`の場合は，末尾に**(x64)**がついているものをダウンロードしてください．
{: .prompt-info }

> テーマとして*Chirpy*を用いる場合は，Ruby+Devkit 3.2.10-1 を選択します．
どうやら，CHirpyは最新バージョンには対応していないようです．
{: .prompt-warning }

ダウンロードしたインストーラを起動し，画面に表示される指示に適宜答えていってください．
指示に答え終わるとインストールが始まります．
最後にコマンドプロンプトが起動すると思いますので，
`1,2,3`と入力し，Enterを押してください．

完了後，以下のコマンドをコマンドプロンプトへ入力し，
バージョンが出力されることを確認してください．
```bash
$ ruby -v
-> ruby 3.2.10 (2026-01-14 revision a3a6d25788) [x64-mingw-ucrt]
```
```bash
$ gem -v
-> 3.4.19
```

## ツールのダウンロード
---
最後に，以下のコマンドを実行しツールをダウンロードしてください．
```bash
$ gem install jekyll bundler
```
ダウンロード後に，以下のコマンドを実行し，ダウンロードされたことを確認してください．
```bash
$ bundle -v
-> 4.0.7
$ bundle exec jekyll -v
-> jekyll 4.4.1
```
